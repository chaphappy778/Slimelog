# SlimeLog Guide — Image Assets

Drop guide images here. They'll be served under `/guide/*` URLs by Next.js.

## Naming convention

Use kebab-case with the slime type or section prefix:

- Base textures (Part One):
  `guide/textures/clear-ultraviolet.jpg`
  `guide/textures/water-fairy-water.jpg`
  `guide/textures/jelly-jellyfish.jpg`
  `guide/textures/butter-mooshake.jpg`
  (matches the `(example …)` reference in the guide copy)

- Add-ins (Part Two): `guide/addins/foam-beads.jpg`
- Containers (Part Three): `guide/containers/screw-lid.jpg`
- Etc — one subfolder per part

Any format works (jpg / png / webp). Aim for 800-1200px on the long edge; ~200-400KB per file after compression is plenty.

## Anti-AI-art rule reminder

Real photos or human-illustrator artwork only. No AI-generated images per the community's expressed preference.
